=============================================
     ŞİFRELEME UYGULAMASI KULLANIM TALİMATLARI
=============================================

Sevgili Kullanıcı,

Bu uygulamayı çalıştırmadan önce aşağıdaki adımları izleyin:

1. PYTHON KURULUMU:
   - Python 3.10 kurulu olmalıdır (https://www.python.org/downloads/)
   - Kurulum sırasında "Add Python to PATH" seçeneğini işaretlediğinizden emin olun

2. PyQt5 KURULUMU:
   - PyQt5 modülünü yüklemek için şu YÖNTEMLERDEN BİRİNİ uygulayın:
   
   a) Çalıştır.bat dosyasına çift tıklayın - otomatik yükleme yapılacaktır
      
   b) Komut isteminde kendiniz yükleyin:
      -> Windows PowerShell veya CMD açın
      -> Şu komutu yazın:  pip install PyQt5==5.15.9
      
   c) Eğer yukarıdaki yöntemler çalışmazsa şu alternatif komutu deneyin:
      pip install --index-url=https://download.qt.io/official_releases/QtForPython/ PyQt5==5.15.9

3. WinRAR KURULUMU:
   - WinRAR programı kurulu olmalıdır (https://www.win-rar.com/download.html)

4. UYGULAMAYI ÇALIŞTIRMA:
   - Tüm gereksinimler yüklendikten sonra "Calistir.bat" dosyasına çift tıklayarak
     programı başlatabilirsiniz.

5. SORUN GİDERME:
   - Hata alırsanız, şu adımları deneyin:
     a) Python'un PATH'e eklendiğinden emin olun
     b) py veya python komutuyla Python'u çalıştırabildiğinizi doğrulayın
     c) pip veya py -m pip komutunu kullanarak PyQt5 yükleyin

NOT: Türkçe karakter sorunu yaşarsanız, SifrelemeHVS.py dosyasındaki 
kodda "utf-8" encoding olduğundan emin olun. 

ÖNEMLİ BİLGİLER VE SORUN GİDERME
===============================

1. Program Gereksinimleri:
   - Python 3.8 veya üstü
   - PyQt5 5.15.9
   - WinRAR (C:\Program Files\WinRAR konumunda)

2. Sık Karşılaşılan Hatalar:
   a) "PyQt5 modülü bulunamadı" hatası:
      - Calistir.bat dosyasını yönetici olarak çalıştırın

   b) "WinRAR bulunamadı" hatası:
      - WinRAR'ı varsayılan konuma kurun
      - Veya programı kapatıp WinRAR'ı kurduktan sonra tekrar açın

   c) Türkçe karakter sorunu:
      - Windows'ta Türkçe dil paketinin kurulu olduğundan emin olun
      - Programı yönetici olarak çalıştırın

3. Şifreleme İşlemi:
   - Metin en az 2 karakter olmalıdır
   - Sayı değeri, seçilen dereceye göre minimum değerden büyük olmalıdır
   - Başlık boş bırakılamaz

4. Çözme İşlemi:
   - Dosya adı formatı: [SatırSayısı][ElementHarf][Derece],[BAST],[SatırSayısı]_[Başlık].rar
   - Örnek: 3T90,1288-1968-2793-1391-1995,3_ProjeOrnegi.rar

5. Güvenlik Notları:
   - Şifreli dosyaları güvenli bir yerde saklayın
   - BAST değerlerini ve şifreleri kimseyle paylaşmayın
   - Programı güvenilir kaynaklardan indirin

6. Performans İpuçları:
   - Büyük metinler için işlem biraz zaman alabilir
   - Program çalışırken başka ağır programlar çalıştırmayın
   - Yeterli disk alanı olduğundan emin olun

7. İletişim ve Destek:
   - Sorun yaşarsanız hata mesajını not alın
   - Programı kapatıp tekrar açmayı deneyin
   - Gerekirse bilgisayarı yeniden başlatın

Not: Bu program sadece eğitim amaçlıdır. Önemli verilerinizi şifrelerken ek güvenlik önlemleri alın.
