ŞİFRELEME PROGRAMI KURULUM TALİMATLARI
=====================================

1. Python Kurulumu:
   - Python 3.8 veya üstü sürümü kurulu olmalıdır
   - https://www.python.org/downloads/ adresinden indirebilirsiniz
   - Kurulum sırasında "Add Python to PATH" seçeneğini işaretleyin

2. WinRAR Kurulumu:
   - WinRAR programı kurulu olmalıdır
   - https://www.win-rar.com/download.html adresinden indirebilirsiniz
   - Varsayılan kurulum konumu: C:\Program Files\WinRAR

3. Program Kurulumu:
   - Tüm dosyaları aynı klasörde tutun
   - Calistir.bat dosyasını çalıştırın
   - Gerekli paketler otomatik olarak kurulacaktır

4. Programı Çalıştırma:
   - Calistir.bat dosyasına çift tıklayın
   - Program otomatik olarak başlayacaktır

5. Sorun Giderme:
   - Python kurulu değilse, Calistir.bat size uyarı verecektir
   - WinRAR kurulu değilse, program size uyarı verecektir
   - Hata durumunda ONEMLI_OKUYUN.txt dosyasını kontrol edin

Not: Programı çalıştırmadan önce Python ve WinRAR'ın kurulu olduğundan emin olun.
